# CustomCSS_Youtube
クソデカサムネイルを改善。  
## ダウンロード
↓のブラウザの拡張機能をダウンロード  
[Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=JA)  
## やり方
youtubeを開いた状態で拡張機能からStylusを選択  
次のスタイルを書く:  
www.youtube.com/  
になってればそこを押して **OK！**  
その後CSSファイルの中身をコピペして保存  
youtubeを更新するとクソデカサムネイルが改善されます。
